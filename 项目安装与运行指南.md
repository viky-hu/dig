## **2. 安装Node.js**

然后，浏览器搜索node.js,下载，跟随步骤安装，有要打勾的直接打勾，直到安装完成。

## **3. 配置环境变量**

win + R 输入`sysdm.cpl`，回车，进入环境变量配置，在系统变量中找到“Path”，点进去，按“新建”，把下面两条，一个个加进去：

```
C:\Program Files\nodejs\
```
```
C:\Users\Administrator\AppData\Roaming\npm
```

## **4. 运行项目**

进入cmd，一条一条输入以下命令——

```
node -v
```

(检查是否安装了node.js)

```
npm -v
```

(检查是否安装了npm)

上面两条没问题的话，继续：

```
npm install -g pnpm
```

下载完之后

```
pnpm -v
```

（检查是否安装了pnpm）

没什么问题的话：

```
cd dig
pnpm install
```

（一会就下好了）

最后：

```
pnpm dev
```

（运行dig文件夹）

打开浏览器进入`http://localhost:3000/`，即可运行本项目
